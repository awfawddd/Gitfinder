# Gitfinder
Uno script che trova le info di qualsiasi utente semplicemente digitando il loro nome GitHub.
## Come si usa
Installa requests:

pip install requests

Puoi eseguirlo con questo comando:

python gitclass.py

Ti mostra:
-nome
-repository
-data di creazione dell'account
-descrizione
-nomi dei repository
-stelle totali
